完美安装paddle、PPOCRLabel、eiseg(linux)

Feb 19, 2022
阅读时长: 2 分钟

中心思想：遇事不决，降低版本 （numpy,paddlepaddle,opencv）

以防万一的操作

1.安装PyQt5

sudo apt-get install python3-pyqt5

2.安装qt-designer

sudo apt-get install qt5-default qttools5-dev-tools

#命令行输入designer可测试是否安装成功

推荐安装流程（从新建虚拟环境到安装一条龙服务）

请严格按照顺序！

#[装错了想卸载重装，把install改成uninstall 不要加包括-i在内后面的信息即可]

#创建虚拟环境

conda create -n 改成你的虚拟环境名字 python=3.7 【3.7别打3.9】

#进入虚拟环境

conda activate 改成刚才你的虚拟环境名字

#装坏了想炸掉虚拟环境（慎重别打错了）

conda remove -n 你的虚拟环境名字 --all

#此时你能看到终端左侧显示环境名，接下来安装各类程序

#注：这里采用百度源作为示范，你也可以使用清华源：

-i https://pypi.tuna.tsinghua.edu.cn/simple

#升级pip

pip install --upgrade pip -i https://mirror.baidu.com/pypi/simple

#以防万一的操作

pip install qtpy -i https://mirror.baidu.com/pypi/simple

pip install pyqt5 -i https://mirror.baidu.com/pypi/simple

#安装paddle github主页面 【这里采用的是cpu，如果要gpu详细安装过程参考gpu怎么看】

（如果出现问题可以在paddlepaddle后面加上==2.1.0安装旧版本）

python -m pip install paddlepaddle -i https://mirror.baidu.com/pypi/simple

#安装PPOCRLabel（OCR） github主页面

pip install PPOCRLabel -i https://mirror.baidu.com/pypi/simple

pip install trash-cli -i https://mirror.baidu.com/pypi/simple

#安装EISeg（图像处理） github主页面

​ pip install eiseg -i https://mirror.baidu.com/pypi/simple #只输入这个就行

​ pip install paddleseg -i https://mirror.baidu.com/pypi/simple

#【最后】安装旧版本opencv（如果装了请先卸载）

pip install opencv-python==4.2.0.32 -i https://mirror.baidu.com/pypi/simple

#接着测试ocr和eiseg：首先输入PPOCRLabel --lang ch

#如果此时出现一段加载过程，最后出现界面，就说明你安装成功了！！

#再次输入： eiseg 这时候大概率已经成功打开！

倒霉人请看这

如果eiseg打开不成功可以尝试曲线方案（先安装这些依赖库）：

(pip不成功就改成conda)

#首先先把paddleseg的整个项目clone到本地 （以下安装还打不开可以这么做）

conda install gdal

pip install qtpy

pip instal pyqt5

pip install easydict

pip install scikit-image

这时候再进入即可！ （如果你还是打不开，就clone后进入eiseg子文件，按照这个帖子运行exe.py ）

最后使用conda deactivate退出虚拟环境！！
